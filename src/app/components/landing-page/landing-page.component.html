Welcome to Novostoic!

A unified web-based interface for the biosynthesis of thermodynamically feasible, carbon and energy-balanced pathways with a way of selecting enzymes to undergo re-engineering for novel reactions steps.

Learn more about the Novostoic workflow below and get started using the tool today.
